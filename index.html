HTML Forms Cheatsheet

Info from Codecademy.

Created by Alice G.

Form Elements

Tag Name Description
<form> Form Used to collect and send information to an external source.
<input> Input Used for multiple interactable inputs.
<textarea> Text Area Creates a text-box for multi-line input. Supports the rows and cols attributes. Needs a closing tag.
<select> Select Creates a dropdown list. A list of choices for the dropdown list can be created using one or more <option> elements. By default, only one <option> can be selected at a time.
<datalist> Datalist Provides users a list of pre-defined options, and allows them to write alternative inputs as well.
<label> Label Provides identification for a specific <input> based on matching values of the <input>‘s id attribute and the <label>‘s for attribute.

Form Input Types

Type Description
Checkbox Render a single checkbox item. Use the same name attribute for multiple checkboxes on the same topic.
Number Allow the user to enter only numbers and a few special characters inside the field.
Checkbox Render a single checkbox item. Use the same name attribute for multiple checkboxes on the same topic.
Range Add a range slider to act as a selector between a minimum and a maximum value. These values are set using the min and max attributes respectively. The slider can be adjusted to move in different steps or increments using the step attribute.
Text Renders a single row input field that users can type text inside.
Radio Button Renders a single radio button. Multiple radio buttons of a related topic are given the same name attribute value. Only a single option can be chosen from a group of radio buttons.
Password Renders a single row input field which allows the user to type censored text inside the field.

Attributes

Attribute Name Description
action Action Tells the form to send the information. A URL is assigned that determines the recipient of the information.
method Method Tells the form what to do with that information once it’s sent. An HTTP verb is assigned to the method attribute that determines the action to be performed.
type="submit" Submit A submit button will be rendered and, by default, will submit the <form> and execute its action.
name Name In order for a form to send data, it needs to be able to put it into key-value pairs. This is achieved by setting the name attribute of the input element. The name will become the key and the value of the input will become the value the form submits corresponding to the key.
required Required Specifies that the field must include a value. Won't allow form submit without this input field filled.
max Maximum Value Specifies the maximum value for a numerical field.
maxlength Maximum Length Specifies the maximum number of characters that can be entered into the field.
pattern Pattern Uses a regular expression to match against (or validate) the value of the <input>, when the form is submitted.
minlength Minimum Length Specifies minimum amount of characters in a text field.
min Minimum Value Specifies minimum value for a numerical field.
